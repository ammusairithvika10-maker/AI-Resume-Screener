AI Resume Screener & Candidate Intelligence Platform

An AI-powered candidate screening platform that automatically analyzes resumes, matches them against job descriptions, ranks candidates, provides explainable insights, visualizes analytics, and stores candidate history.

Features

- Resume PDF Upload
- PDF Text Extraction
- Semantic Similarity Matching
- Skill Match Analysis
- Candidate Ranking
- AI Ranking Explanation
- Analytics Dashboard
- Plotly Visualization
- Excel Report Export
- SQLite Candidate History

Tech Stack

- Python
- Streamlit
- Sentence Transformers
- Scikit-Learn
- Plotly
- SQLite
- PyMuPDF

Future Enhancements

- Conversational RAG Chatbot
- PDF Report Export
- Advanced HR Analytics
- LLM-Based Explainability
